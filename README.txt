Сборка Apk-файла
Выполнить в командной строке:
gradlew assembleDebug

Руководство пользователя:
Для просмотра товаров необходимо кликнуть на один из предложенных товаров в главном окне. После этого
откроется страница соответствующего товара. 
Часть функций приложения недоступно до авторизации пользователя.
Для этого нужно нажать кнопку login и ввести логин и пароль пользователя, затем нажать кнопку "Войти". Если
у пользователя еще нет аккаунта, его можно создать, нажав кнопку "Регистрация" в главном окне и заполнив
все поля. 
После этого появится возможность добавления товаров в корзину. Для этого необходимо нажать кнопку 
"Добавить в корзину" на странице товара. Чтобы перейти в корзину, в главном окне нужно нажать на кнопку
"Корзина" в правом верхнем углу. После этого откроется страница с товарами. Для удаления товаров из корзины 
необходимо нажать на соответствующую кнопку в окне товара. 
Для оплаты товаров в окне корзины нужно нажать кнопку "Оплатить". После этого откроется активность 
с заполнением данных о банковской карте пользователя, и при нажатии кнопки оплаты на этой странице откроется
окно оплаты при помощи сервиса Google-Pay
